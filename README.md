# ✅ Todo App (Redux Toolkit)

## 🚀 Description
A task management application built using React and Redux Toolkit to demonstrate efficient state management in a scalable frontend application.

## 🔥 Features
- Add, delete, and manage tasks  
- Centralized state management using Redux Toolkit  
- Clean and responsive UI  
- Optimized performance with structured state flow  

## 🛠 Tech Stack
- React.js  
- Redux Toolkit  
- JavaScript  
- HTML, CSS  

## ⚙️ Setup
1. Clone the repository  
2. Run `npm install`  
3. Run `npm start`  

## 🌐 Live Demo
https://todo-rtk-app.onrender.com  

## 🎯 Purpose
This project was built to understand and implement advanced state management using Redux Toolkit and improve application scalability and maintainability.
